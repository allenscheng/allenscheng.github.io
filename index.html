My capstone project while attending Actualize.

With these public deals, users are able to save it to a wish list or visit the website that sells that particular item. The app also allows a photo to be uploaded to help identify the name of a product and where it’s sold using the Google Cloud Vision API.

  • Date: January 2018
  • Tools Used: Javascript, VueJS, Ruby, Rails

RGB

A CSS game I created as a side project.

RGB is a game that displays a randomly selected CSS hex color code. Your objective is to select the correct one with the given color code. If the selection is wrong, the box will disappear. If the selection is correct, all missing boxes will reappear and change to the correct color. There are two modes, easy with 3 boxes and hard with 6 boxes.

  • Date: March 2018
  • Tools Used: Javascript and CSS
